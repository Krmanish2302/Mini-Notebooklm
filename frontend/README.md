# Mini NotebookLM — Frontend

This is the React frontend for Mini NotebookLM.

## Setup

```bash
npm install
npm run dev
```

## Entry Point

`src/App.jsx` — Main application component.  
Connects to the FastAPI backend at the URL configured in `src/api.js` (to be created in Phase 4).
